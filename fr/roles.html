Une app, trois expériences

Conçu pour les clients, magasins et agents.

Waloo s'adapte à chaque rôle tout en gardant la même interface épurée.

Client

Découvrir, réserver, récupérer.

Parcourez les offres sur la carte, ajustez le rayon, filtrez par catégorie. Réservez en un tap et récupérez avec votre code OTP.

Magasin

Lister le surplus, gérer les retraits.

Créez des offres avec prix, fenêtres de retrait et stock. Confirmez les retraits via OTP. Suivez vos revenus et avis clients.

Agent

Activer le cash-in partout.

Les agents rechargent les crédits en espèces pour les utilisateurs et magasins. Ils rendent l'écosystème accessible là où le paiement digital est rare.
